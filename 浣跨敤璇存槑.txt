WeRead KOReader 扫码登录测试补丁（覆盖文件）

适用基础版本：你本次上传的修改版，main.lua 显示 v0.2.2。
本补丁不修改版本号。

包含文件：
- main.lua
- config.example.lua
- lib/client.lua
- lib/settings.lua
- lib/i18n.lua

安装：
1. 先备份 Kindle 中现有的 weread.koplugin 文件夹。
2. 将本补丁中的文件按原目录覆盖到现有 weread.koplugin。
3. 不要把 config.example.lua 改名，也不要删除现有 config.lua。
4. 完全退出并重新启动 KOReader。

首次测试：
微信读书 → 设置 → 账号管理 → 微信扫码登录

用微信扫描 Kindle 屏幕二维码并在手机确认。成功后应提示：
- 账号名称
- Cookie 已配置
- 官方 API Key 已配置

随后进入“账号状态”确认，再打开“书架”。

当前范围：
- 已实现普通书籍所需的扫码登录、Cookie 保存和 API Key 自动获取。
- 公众号仍可能需要 wr_ticket / x-wrpa-0，暂未纳入本次测试。
- 阅读时长在完全没有旧 curl_payload 时是否稳定上报，仍需后续单独验证。

安全：
不要截图或发送完整 API Key、wr_skey、wr_rt、accessToken、refreshToken。
